------------------------------------------------------------------------------------------
리눅스 배포판
------------------------------------------------------------------------------------------
-------------------------------------------------------------------
1. 데비안 계열
-------------------------------------------------------------------
-> 우분투(데비안에서 파생되어 나옴)
-> 사용자의 편의성에 중점, 설치와 관리가 용이하다
-> 개인용 데스크탑에 적합한 리눅스 배포판
-------------------------------------------------------------------
2. 슬랙웨어 계열
-------------------------------------------------------------------
-> 가장 오래된 배보판, 초창기 배포판, 설치와 관리가 매우어려움
-> 단독으로 사용하는 경우는 없다. (다른 배보판과 혼합하여 사용)
 				  ex) Back Track(+우분투)
				      Kali Linux(+데비안)

-------------------------------------------------------------------
3. 래드햇 계열
-------------------------------------------------------------------
-------------------------------------------------
1,RGEL
-------------------------------------------------
-> Red Hat Enterprise Linux
-> 래드햇에서 만든 기업용 리눅스
-> 기업의 서버용으로 사용
-> 배포는 무료!, 기술지원 유료
-------------------------------------------------
2.Fedora
-------------------------------------------------
-> 최신 패키지 다수 포함 되어있는 리눅스배포판
-> 테스트용, 교육용 리눅스

-------------------------------------------------
3.CentOS
-------------------------------------------------
※(현재 중견기업 이상급에서 자주 사용함)
-> Communit ENTerprise OS
-> 배포무료, 기술지원 X
-> 커뮤니티를 이용해서 기업간 정보교환으로 해결
